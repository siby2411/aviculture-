# 🐔 Aviculture - Omega Consulting

Système de gestion d'élevage de poulets de chair et poules pondeuses.

## Fonctionnalités
- 📊 Tableau de bord avec statistiques en temps réel
- 🐓 Gestion des lots (création, suivi, détails)
- 💰 Gestion des ventes et facturation
- 💳 Suivi des charges et dépenses
- 📋 Rapports d'exploitation et analyse financière
- 🏥 Module de suivi prophylactique (Santé)
- 📸 Galerie d'images des poulets
- 📈 Graphiques et statistiques avancés

## Technologies
- PHP 8+
- MariaDB/MySQL
- HTML5/CSS3
- JavaScript (Chart.js)

## Identité visuelle
- Omega Consulting
- Couleurs: Vert prairie (#2D6A4F) + Ambre doré (#F4A261)

## Installation
1. Cloner le dépôt
2. Configurer la base de données
3. Lancer le serveur PHP
